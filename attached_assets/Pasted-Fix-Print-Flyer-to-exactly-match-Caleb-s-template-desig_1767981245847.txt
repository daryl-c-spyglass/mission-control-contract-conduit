## Issue 1: Replace Emoji Icons with Proper SVG Line Icons

Current: Using emojis for bed/bath/sqft icons - looks unprofessional
Reference: Ad_Graphics_Square_Annotation shows clean line-style icons labeled "BACKGROUND (ICONS)"

The template uses these specific icon styles:
- Bed icon: Simple bed outline with headboard (see left column in PDF)
- Bath icon: Bathtub with claw feet style (see left column in PDF)
- Sqft icon: Rectangle/square with diagonal line representing area

Replace emoji icons with canvas-drawn SVG-style line icons:
```javascript
// Bed icon - matches template style
function drawBedIcon(ctx, x, y, size) {
  ctx.strokeStyle = '#333';
  ctx.lineWidth = 2;
  ctx.beginPath();
  // Headboard (left side, tall)
  ctx.moveTo(x, y + size);
  ctx.lineTo(x, y);
  ctx.lineTo(x + size * 0.2, y);
  ctx.lineTo(x + size * 0.2, y + size * 0.4);
  // Bed surface
  ctx.moveTo(x, y + size * 0.5);
  ctx.lineTo(x + size, y + size * 0.5);
  // Footboard (right side, shorter)
  ctx.lineTo(x + size, y + size * 0.7);
  ctx.moveTo(x + size, y + size * 0.5);
  ctx.lineTo(x + size, y + size);
  ctx.stroke();
}

// Bath icon - bathtub with legs (matches template)
function drawBathIcon(ctx, x, y, size) {
  ctx.strokeStyle = '#333';
  ctx.lineWidth = 2;
  ctx.beginPath();
  // Faucet
  ctx.moveTo(x + size * 0.15, y);
  ctx.lineTo(x + size * 0.15, y + size * 0.3);
  ctx.lineTo(x + size * 0.3, y + size * 0.3);
  // Tub rim
  ctx.moveTo(x, y + size * 0.35);
  ctx.lineTo(x + size, y + size * 0.35);
  // Tub body
  ctx.moveTo(x + size * 0.05, y + size * 0.35);
  ctx.lineTo(x + size * 0.05, y + size * 0.75);
  ctx.quadraticCurveTo(x + size * 0.05, y + size * 0.85, x + size * 0.15, y + size * 0.85);
  ctx.lineTo(x + size * 0.85, y + size * 0.85);
  ctx.quadraticCurveTo(x + size * 0.95, y + size * 0.85, x + size * 0.95, y + size * 0.75);
  ctx.lineTo(x + size * 0.95, y + size * 0.35);
  // Legs
  ctx.moveTo(x + size * 0.15, y + size * 0.85);
  ctx.lineTo(x + size * 0.1, y + size);
  ctx.moveTo(x + size * 0.85, y + size * 0.85);
  ctx.lineTo(x + size * 0.9, y + size);
  ctx.stroke();
}

// Square footage icon - rectangle with diagonal (area symbol)
function drawSqftIcon(ctx, x, y, size) {
  ctx.strokeStyle = '#333';
  ctx.lineWidth = 2;
  ctx.beginPath();
  // Rectangle
  ctx.rect(x, y + size * 0.1, size, size * 0.7);
  // Diagonal arrow
  ctx.moveTo(x + size * 0.2, y + size * 0.6);
  ctx.lineTo(x + size * 0.8, y + size * 0.25);
  // Arrow head
  ctx.lineTo(x + size * 0.65, y + size * 0.25);
  ctx.moveTo(x + size * 0.8, y + size * 0.25);
  ctx.lineTo(x + size * 0.8, y + size * 0.4);
  ctx.stroke();
}
```
