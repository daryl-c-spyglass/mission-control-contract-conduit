## Problem

The **Spyglass Realty logo** in the footer of CMA Presentation PDF export is **unreadable/unclear**.

---

## Possible Causes

1. **Logo too small** - Logo scaled down too much for footer
2. **Low resolution image** - Source logo file is low quality
3. **Dark logo on dark background** - Contrast issue
4. **Compression artifacts** - PDF generation compressing the image
5. **Wrong logo variant** - Using wrong version (e.g., full color on dark bg)

---

## Fix Options

### Option 1: Use Higher Resolution Logo

```tsx
// Use a higher resolution version of the logo
// Minimum 300 DPI for print quality

// Instead of:
<img src="/logos/spyglass-logo.png" />

// Use:
<img src="/logos/spyglass-logo-hd.png" /> // Higher resolution version
// OR
<img src="/logos/spyglass-logo.svg" />   // SVG scales perfectly
```

### Option 2: Increase Logo Size in Footer

```tsx
// Current (too small):
<img src={logo} className="h-4" />

// Fixed (larger):
<img src={logo} className="h-8" /> // or h-6 minimum
```

### Option 3: Use Appropriate Logo Variant

```tsx
// For dark backgrounds:
<img src="/logos/spyglass-logo-white.png" />

// For light backgrounds:
<img src="/logos/spyglass-logo-dark.png" />

// Or use the full color version that works on both:
<img src="/logos/spyglass-logo-color.png" />
```

### Option 4: Add Background/Padding for Contrast

```tsx
// Add white background behind logo for visibility
<div className="bg-white px-2 py-1 rounded">
  <img src={logo} className="h-6" alt="Spyglass Realty" />
</div>
```

### Option 5: Ensure Logo Loads in PDF Generation

```tsx
// When using Puppeteer for PDF generation, ensure images are loaded:

// Convert logo to base64 to embed directly in PDF
const logoBase64 = await imageToBase64('/logos/spyglass-logo.png');

// Use in slide:
<img src={`data:image/png;base64,${logoBase64}`} className="h-8" />
```

---

## Recommended Fix

1. **Use SVG logo** if available (scales perfectly at any size)
2. **Minimum height of 24px (h-6)** for footer logos
3. **Ensure proper contrast** (white logo on dark footer, or add white bg)
4. **Convert to base64** for PDF generation to prevent loading issues
