### 2. LOGO AUDIT

Find all logo references and usages:

```bash
# Find all logo files
find . -type f \( -name "*logo*" -o -name "*spyglass*" -o -name "*Logo*" \) -not -path "./node_modules/*"

# Find all logo imports/references in code
grep -r "logo" --include="*.tsx" --include="*.ts" . | grep -v node_modules

# Find image src attributes that might contain logos
grep -rE "src=.*logo|src=.*spyglass|src=.*Logo" --include="*.tsx" --include="*.ts" . | grep -v node_modules
```

**Check each logo usage for:**
- [ ] Is it using an official logo format (not custom/altered)?
- [ ] Is the correct version used for the background? (white logo on dark bg, black logo on light bg)
- [ ] Are any logos being rendered as text instead of images?
- [ ] Are logo colors being altered via CSS filters or opacity changes?
