Two improvements needed:

### Part 1: Add Consistent Format Badges to Marketing Assets Gallery

**Current State:**
- Print Flyer: Has orange badge ✅
- Facebook 16:9: Plain text label ❌

**Required:**
All asset types should have colored badges for visual consistency.

**Badge Colors:**
```
Print Flyer    → Orange (#f97316) background, white text
Social Square  → Blue (#3b82f6) background, white text  
Social 16:9    → Purple (#8b5cf6) background, white text
Story 9:16     → Pink (#ec4899) background, white text
```

**Implementation:**
```tsx
const formatBadgeStyles: Record<string, string> = {
  'print': 'bg-orange-500 text-white',
  'square': 'bg-blue-500 text-white',
  'landscape': 'bg-purple-500 text-white',  // 16:9
  'story': 'bg-pink-500 text-white',         // 9:16
};

// In the asset card:
<Badge className={formatBadgeStyles[asset.format] || 'bg-gray-500 text-white'}>
  {formatLabels[asset.format]}
</Badge>
```

**Format Labels:**
```
print     → "Print Flyer"
square    → "Instagram Post"
landscape → "Facebook 16:9"
story     → "Instagram Story"
```
