## Issue 1: Address Number Spacing

Current: "1 3 1 0 6  N E W  B O S T O N" (numbers spaced individually)
Should be: "13106  N E W  B O S T O N" (numbers together, only letters spaced)

Fix the address formatting:
```javascript
function formatAddressForFlyer(address: string): string {
  return address
    .toUpperCase()
    .split(' ')
    .map(word => {
      // Keep numbers together (street number)
      if (/^\d+$/.test(word)) {
        return word;
      }
      // Space out letters for text words
      return word.split('').join(' ');
    })
    .join('   '); // Triple space between words
}
// Result: "13106   N E W   B O S T O N"
```
