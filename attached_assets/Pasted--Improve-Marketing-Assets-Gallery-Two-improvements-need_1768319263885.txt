### Issue 2: Add Click-to-Preview Modal

**Requirement:**
When user clicks on a marketing asset card, open a modal/floating card showing:
- Large preview of the asset
- Asset details (type, format, date created)
- Action buttons (Edit, Download, Delete, Share to Slack)

**Implementation:**

Create a preview modal component:

```tsx
interface AssetPreviewModalProps {
  asset: MarketingAsset;
  isOpen: boolean;
  onClose: () => void;
  onEdit: () => void;
  onDownload: () => void;
  onDelete: () => void;
}

function AssetPreviewModal({ 
  asset, 
  isOpen, 
  onClose, 
  onEdit, 
  onDownload, 
  onDelete 
}: AssetPreviewModalProps) {
  return (
    <Dialog open={isOpen} onOpenChange={onClose}>
      <DialogContent className="max-w-4xl">
        {/* Large Image Preview */}
        <div className="flex justify-center bg-gray-100 rounded-lg p-4">
          <img 
            src={asset.imageUrl} 
            alt={asset.type}
            className="max-h-[70vh] object-contain"
          />
        </div>
        
        {/* Asset Info */}
        <div className="flex items-center justify-between mt-4">
          <div>
            <Badge>{asset.format}</Badge>
            <p className="text-sm text-muted-foreground mt-1">
              Created {formatDate(asset.createdAt)}
            </p>
          </div>
          
          {/* Action Buttons */}
          <div className="flex gap-2">
            <Button variant="outline" onClick={onEdit}>
              <Pencil className="h-4 w-4 mr-2" />
              Edit
            </Button>
            <Button variant="outline" onClick={onDownload}>
              <Download className="h-4 w-4 mr-2" />
              Download
            </Button>
            <Button variant="destructive" onClick={onDelete}>
              <Trash className="h-4 w-4 mr-2" />
              Delete
            </Button>
          </div>
        </div>
      </DialogContent>
    </Dialog>
  );
}
```
