### Dimensions

### Header Section (top ~300px)

- Left side: Spyglass Realty logo (white version)

- Right side: Price badge

### Address Bar (~100px below header)
- Full address in UPPERCASE with SPACED LETTERS
- Example: "1 3 1 0 6   N E W   B O S T O N,   A U S T I N,   T X   7 8 7 2 9"
- Centered, dark text on white background

### Photos Section (~1800px)
- Main photo: Full width, approximately 60% of photo area height
- Two secondary photos below: 
  - Side by side, equal width (50% each)
  - Approximately 40% of photo area height
- All photos: object-fit cover, no gaps between them

### Bottom Info Section (~800px) - Three Columns

**Left Column - Property Stats (with icons):**
- Bed icon + "X bedrooms"
- Bath icon + "X bathrooms"
- Square footage icon + "X,XXX sq. ft"
- Stack vertically with spacing
- Use simple line icons

**Center Column - Description:**
- Listing Headline (if provided) - uppercase, spaced letters, bold
  - Example: "A PRIME OPPORTUNITY IN NORTH EAST AUSTIN"
- Property description below (115 char max, truncated at word boundary)
- Text aligned left or justified

**Right Column - Agent Info:**
- Agent photo (circular, ~150px diameter)
- Team logo placeholder (small Spyglass logo)
- Agent name (bold, larger text)
- Agent title "REALTOR®, Spyglass Realty"

### Background Decorations
