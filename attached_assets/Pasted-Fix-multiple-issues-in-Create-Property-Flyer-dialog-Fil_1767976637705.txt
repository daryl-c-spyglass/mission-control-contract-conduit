## Issue 5: Print Flyer Address Spacing

The address bar shows incorrect spacing. Current: "1 3 1 0 6 N E W B O S T O N"
Should have proper spacing between words, not just letters.

Fix the address formatting function:
```javascript
function formatAddressSpaced(address: string): string {
  // Split into words first, then space letters within each word
  // Add extra space between words
  return address
    .toUpperCase()
    .split(' ')
    .map(word => word.split('').join(' '))
    .join('   '); // Triple space between words
}

// Example output: "1 3 1 0 6   N E W   B O S T O N"
// Or even better, keep numbers together:
// "13106   N E W   B O S T O N"
```

Better approach - only space the letters, keep numbers grouped:
```javascript
function formatAddressSpaced(address: string): string {
  return address
    .toUpperCase()
    .split(' ')
    .map(word => {
      // If word is all numbers (street number), keep together
      if (/^\d+$/.test(word)) {
        return word;
      }
      // Otherwise space the letters
      return word.split('').join(' ');
    })
    .join('   '); // Triple space between words
}

// Output: "13106   N E W   B O S T O N"
```
