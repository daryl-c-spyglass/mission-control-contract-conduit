Update Sidebar User Profile Display

Task: Display Logged-In User's Info in Sidebar
The lower left corner of the sidebar currently shows hardcoded "Agent User" and "agent@realty.com". Update this to dynamically show the actual logged-in user's information.
Requirements

Display User's Actual Information:

Profile Photo: From Agent Marketing Profile (Settings) or Google profile photo as fallback
Display Name: From Agent Marketing Profile, or Google name, or email username as fallback
Email: From logged-in user's authentication email


Data Priority (fallback chain):
For Name:

   1. Agent Marketing Profile â†’ Display Name (if set)
   2. Google OAuth name (from login)
   3. Email username (part before @)
For Photo:
   1. Agent Marketing Profile â†’ Photo (if uploaded)
   2. Google OAuth profile picture
   3. Default avatar/initials
For Email:
   1. Logged-in user's email (from auth session)

Implementation

tsx   // In Sidebar component
   const { user } = useAuth(); // Get logged-in user from auth context
   const { marketingProfile } = useMarketingProfile(); // Get Agent Marketing Profile
   
   const displayName = marketingProfile?.displayName || user?.name || user?.email?.split('@')[0] || 'Agent';
   const displayEmail = user?.email || '';
   const displayPhoto = marketingProfile?.photo || user?.profilePicture || null;
   
   // In the sidebar footer
   <div className="sidebar-user-profile">
     <div className="user-avatar">
       {displayPhoto ? (
         <img src={displayPhoto} alt={displayName} className="avatar-image" />
       ) : (
         <div className="avatar-initials">
           {displayName.charAt(0).toUpperCase()}
         </div>
       )}
     </div>
     <div className="user-info">
       <span className="user-name">{displayName}</span>
       <span className="user-email">{displayEmail}</span>
     </div>
   </div>
