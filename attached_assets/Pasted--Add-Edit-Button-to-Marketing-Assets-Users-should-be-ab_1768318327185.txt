## Add Edit Button to Marketing Assets

Users should be able to edit previously created marketing assets, not just download or delete them.

### Changes to make:

**1. Add Edit button to asset cards in Marketing Assets gallery**

Location: Marketing tab asset cards (next to download button)

Current buttons: [Download] [Delete]
New buttons: [Edit] [Download] [Delete]

The Edit button should:
- Open the appropriate dialog pre-filled with the asset's data
- For "Print Flyer" assets â†’ Open CreateFlyerDialog with saved data
- For "Social" assets â†’ Open MarketingMaterialsDialog with saved data

**6. Asset card Edit button implementation**

```tsx
// In the asset card component
<Button
  variant="ghost"
  size="icon"
  onClick={() => {
    if (asset.type === 'flyer') {
```
