## Enhance Templates Tab - Aligned with Create Graphics & Create Flyer

The Templates tab needs to integrate seamlessly with the existing "Create Graphics" and "Create Flyer" features in the Marketing tab.

---

## Current Architecture

```
Marketing Tab:
├── Marketing Assets Gallery (view created assets)
├── [Create Graphics] button → Opens dialog for social posts
└── [Create Flyer] button → Opens dialog for print flyers

Templates Tab:
├── Search for listing
├── Posts | Stories | Flyers | Post Cards | Brochures tabs
└── Template previews
```

**Problem:** Overlap and confusion between Marketing and Templates tabs.

---

## Unified Architecture (Recommended)

```
Marketing Tab:
├── Marketing Assets Gallery (view/manage created assets)
├── [Create Graphics] → Quick create with current listing
└── [Create Flyer] → Quick create with current listing

Templates Tab (Template Library):
├── Auto-loads current transaction listing
├── Browse ALL template designs
├── Categories: Social | Print | Digital
├── Click template → Opens SAME dialog as Create Graphics/Create Flyer
│   └── Pre-selects the chosen template design
└── Created assets saved to Marketing tab gallery
```

**Key Insight:** Templates tab is the **browsing/discovery** experience. Create Graphics and Create Flyer are the **creation** tools. They should use the SAME underlying components.

### 2. Unified Creation Dialog

Create a single dialog component that handles both Create Graphics and Create Flyer, accepting an optional template parameter:

```tsx
// components/create-marketing-material-dialog.tsx

interface CreateMarketingMaterialDialogProps {
  open: boolean;
  onOpenChange: (open: boolean) => void;
  transaction: Transaction;
  initialTemplate?: TemplateDesign;  // Pre-selected from Templates tab
  initialCategory?: 'social' | 'print';  // From Create Graphics vs Create Flyer button
}

function CreateMarketingMaterialDialog({
  open,
  onOpenChange,
  transaction,
  initialTemplate,
  initialCategory = 'social',
}: CreateMarketingMaterialDialogProps) {
  const [category, setCategory] = useState(initialTemplate?.category || initialCategory);
  const [selectedTemplate, setSelectedTemplate] = useState<TemplateDesign | null>(initialTemplate || null);
  const [selectedFormat, setSelectedFormat] = useState(initialTemplate?.subcategory || 'posts');
  const [selectedPhoto, setSelectedPhoto] = useState(transaction.mlsData?.images?.[0] || '');
  const [status, setStatus] = useState(getDefaultStatus(transaction.status));
  const [description, setDescription] = useState('');

  // Get available templates based on category
  const availableTemplates = getTemplatesByCategory(category);
  const formatOptions = getFormatOptions(category);

  return (
    <Dialog open={open} onOpenChange={onOpenChange}>
      <DialogContent className="max-w-5xl max-h-[90vh] overflow-y-auto">
        <DialogHeader>
          <DialogTitle>
            {category === 'social' ? 'Create Marketing Graphics' : 'Create Print Material'}
          </DialogTitle>
          <DialogDescription>
            Create professional marketing materials for {transaction.address}
          </DialogDescription>
        </DialogHeader>

        <div className="grid grid-cols-2 gap-6">
          {/* Left: Options */}
          <div className="space-y-6">
            {/* Category Toggle (if not locked) */}
            {!initialTemplate && (
              <div>
                <Label>Material Type</Label>
                <div className="flex gap-2 mt-2">
                  <Button
                    variant={category === 'social' ? 'default' : 'outline'}
                    onClick={() => setCategory('social')}
                  >
                    <Instagram className="h-4 w-4 mr-2" />
                    Social Media
                  </Button>
                  <Button
                    variant={category === 'print' ? 'default' : 'outline'}
                    onClick={() => setCategory('print')}
                  >
                    <Printer className="h-4 w-4 mr-2" />
                    Print
                  </Button>
                </div>
              </div>
            )}

            {/* Format Selection */}
            <div>
              <Label>Format</Label>
              <div className="grid grid-cols-3 gap-2 mt-2">
                {formatOptions.map((format) => (
                  <button
                    key={format.id}
                    onClick={() => setSelectedFormat(format.id)}
                    className={cn(
                      "flex flex-col items-center p-3 rounded-lg border-2 transition-all",
                      selectedFormat === format.id
                        ? "border-primary bg-primary/10"
                        : "border-border hover:border-primary/50"
                    )}
                  >
                    <div className={cn(
                      "border-2 border-current mb-2",
                      format.aspectClass // e.g., "w-8 h-8" for 1:1
                    )}>
                      {format.ratio}
                    </div>
                    <span className="text-sm">{format.name}</span>
                  </button>
                ))}
              </div>
            </div>

            {/* Template Style Selection */}
            <div>
              <Label>Template Style</Label>
              <div className="grid grid-cols-4 gap-2 mt-2">
                {availableTemplates
                  .filter(t => t.subcategory === selectedFormat)
                  .map((template) => (
                    <button
                      key={template.id}
                      onClick={() => setSelectedTemplate(template)}
                      className={cn(
                        "rounded-lg overflow-hidden border-2 transition-all",
                        selectedTemplate?.id === template.id
                          ? "border-primary ring-2 ring-primary/20"
                          : "border-transparent hover:border-primary/50"
                      )}
                    >
                      <img src={template.previewUrl} alt={template.name} className="w-full" />
                    </button>
                  ))}
              </div>
            </div>

            {/* Photo Selection */}
            <div>
              <div className="flex items-center justify-between">
                <Label>Property Photo</Label>
                {transaction.mlsData?.imageInsights && (
                  <Badge variant="secondary" className="text-xs">
                    <Sparkles className="h-3 w-3 mr-1" />
                    AI Recommended
                  </Badge>
                )}
              </div>
              <div className="grid grid-cols-4 gap-2 mt-2">
                {transaction.mlsData?.images?.slice(0, 8).map((photo, i) => (
                  <button
                    key={i}
                    onClick={() => setSelectedPhoto(photo)}
                    className={cn(
                      "aspect-square rounded overflow-hidden border-2",
                      selectedPhoto === photo ? "border-primary" : "border-transparent"
                    )}
                  >
                    <img src={photo} className="w-full h-full object-cover" />
                  </button>
                ))}
              </div>
            </div>

            {/* Status Selection */}
            <div>
              <Label>Status Badge</Label>
              <Select value={status} onValueChange={setStatus}>
                <SelectTrigger className="mt-2">
                  <SelectValue />
                </SelectTrigger>
                <SelectContent>
                  {getStatusOptions(transaction.status).map((s) => (
                    <SelectItem key={s} value={s}>{s}</SelectItem>
                  ))}
                </SelectContent>
              </Select>
            </div>

            {/* Description (for social) */}
            {category === 'social' && (
              <div>
                <div className="flex items-center justify-between">
                  <Label>Description</Label>
                  <Button variant="ghost" size="sm" onClick={handleAISuggest}>
                    <Sparkles className="h-4 w-4 mr-1" />
                    AI Suggest
                  </Button>
                </div>
                <Input
                  value={description}
                  onChange={(e) => setDescription(e.target.value)}
                  maxLength={80}
                  placeholder="e.g. Stunning 4BR with Chef's Kitchen"
                  className="mt-2"
                />
              </div>
            )}
          </div>

          {/* Right: Live Preview */}
          <div className="bg-muted rounded-lg p-4 flex items-center justify-center">
            <div className={cn(
              "relative bg-white shadow-lg",
              getPreviewAspectClass(selectedFormat)
            )}>
              <MaterialPreview
                template={selectedTemplate}
                photo={selectedPhoto}
                status={status}
                listing={transaction.mlsData}
                description={description}
              />
            </div>
          </div>
        </div>

        <DialogFooter className="mt-6">
          <Button variant="outline" onClick={() => onOpenChange(false)}>
            Cancel
          </Button>
          <Button onClick={handleGenerate}>
            <Download className="h-4 w-4 mr-2" />
            Generate & Download
          </Button>
          <Button onClick={handleSaveToMarketing} variant="secondary">
            <Save className="h-4 w-4 mr-2" />
            Save to Marketing
          </Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```

```tsx
function MarketingTab({ transaction }: { transaction: Transaction }) {
  const [createDialogOpen, setCreateDialogOpen] = useState(false);

        </div>

        transaction={transaction}

    </div>
```

```tsx
// Templates tab - browses templates, opens same dialog with pre-selected template
function TemplatesTab({ transaction }: { transaction: Transaction }) {
  const [createDialogOpen, setCreateDialogOpen] = useState(false);
  const [selectedTemplate, setSelectedTemplate] = useState<TemplateDesign | null>(null);
  const [category, setCategory] = useState<'social' | 'print' | 'digital'>('social');
  const [subcategory, setSubcategory] = useState<string>('posts');
  const [statusFilter, setStatusFilter] = useState<string>('all');

  // Auto-load current transaction (no search needed)
  const listing = transaction.mlsData;

  const handleTemplateClick = (template: TemplateDesign) => {
    setSelectedTemplate(template);
    setCreateDialogOpen(true);
  };

  // Filter templates
  const filteredTemplates = templateDesigns.filter(t => {
    if (t.category !== category) return false;
    if (subcategory !== 'all' && t.subcategory !== subcategory) return false;
    return true;
  });

  return (
    <div className="space-y-6">
      {/* Header */}
      <div>
        <h2 className="text-2xl font-bold">Template Library</h2>
        <p className="text-muted-foreground">

        <CardContent className="flex items-center gap-4 py-4">

            <img
              src={listing.images[0]}
              alt="Property"
              className="w-20 h-16 object-cover rounded"
            />
          )}

            <p className="font-semibold">{transaction.address}</p>
            <p className="text-sm text-muted-foreground">
              ${listing?.listPrice?.toLocaleString()} • {listing?.details?.numBedrooms} bed • {listing?.details?.numBathrooms} bath

          </div>
          <Badge className={getStatusBadgeStyle(transaction.status)}>

          <div className="flex items-center gap-3">
            <div className="p-2 bg-primary/10 rounded-lg">
              <Wand2 className="h-5 w-5 text-primary" />
            </div>
            <div>
              <p className="font-medium">Quick Create All Formats</p>
              <p className="text-sm text-muted-foreground">
                Generate Instagram Post + Facebook Post + Story with one click
              </p>
            </div>
          </div>
          <Button onClick={handleQuickCreateAll}>
            <Sparkles className="h-4 w-4 mr-2" />
            Generate All
          </Button>
        </CardContent>
      </Card>

      {/* Category Tabs */}
      <Tabs value={category} onValueChange={(v) => setCategory(v as any)}>
        <TabsList>
          <TabsTrigger value="social" className="gap-2">
            <Instagram className="h-4 w-4" />
            Social Media
          </TabsTrigger>
          <TabsTrigger value="print" className="gap-2">
            <Printer className="h-4 w-4" />
            Print Materials
          </TabsTrigger>
          <TabsTrigger value="digital" className="gap-2">
            <Mail className="h-4 w-4" />
            Digital
          </TabsTrigger>
        </TabsList>

        <TabsContent value="social" className="space-y-4">
          {/* Sub-filters */}
          <div className="flex items-center gap-4">
            <div className="flex gap-2">
              {['all', 'posts', 'stories', 'facebook'].map((sub) => (
                <Badge
                  key={sub}
                  variant={subcategory === sub ? 'default' : 'outline'}
                  className="cursor-pointer"
                  onClick={() => setSubcategory(sub)}
                >
                  {sub === 'all' ? 'All' : sub.charAt(0).toUpperCase() + sub.slice(1)}
                </Badge>
              ))}
            </div>
            <Select value={statusFilter} onValueChange={setStatusFilter}>
              <SelectTrigger className="w-40">

                <SelectItem value="under-contract">Under Contract</SelectItem>

              </SelectContent>

          {/* Template Grid */}
          <div className="grid grid-cols-2 md:grid-cols-4 gap-4">
            {filteredTemplates.map((template) => (

                key={template.id}
```
